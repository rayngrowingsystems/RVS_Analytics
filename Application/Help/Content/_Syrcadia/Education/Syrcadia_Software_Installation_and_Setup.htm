Syrcadia Software Installation and Setup

RAYN Syrcadia software is a powerful companion to your Alina system, offering the flexibility to configure spectra, day plans, and crop plans to your schedule. These instructions are provided to accompany the provided configuration file as the quickest method to get started with Syrcadia and your red lettuce experiment.

For more information, see the RAYN Syrcadia Software and RAYN Touch Controller User Manual. User documentation and technical support are available via our website, rayngrowingsystems.com, or by contacting your RAYN Growing Systems provider.

Installing Syrcadia

RAYN Syrcadia software is provided on your Alina Educational Horticulture Package USB memory stick. Follow the instructions below to install or update the software on a compatible Windows PC.

  1. If the software has been provided as a zipped archive, unzip the file.
  2. Double-click the .exe installer. The RAYN Syrcadia Setup window will appear.
  3. Follow the Setup prompts and click the Install button to begin software installation.
  4. Select Finish when prompted. RAYN Syrcadia is now ready to use on your PC.

Note: As Syrcadia PC installations will not uninstall or overwrite older versions, it is possible to have multiple versions installed. Users may uninstall versions as desired using Windows Settings.

Loading the Configuration

Once the RAYN Syrcadia software is installed, follow the steps below to load and prepare the provided Red Lettuce First Experiment configuration file.

  1. Select the gear icon in the menu on the left to access Settings.

  2. Select the Files tab.

  3. Select the Open button. A File Explorer window will pop up.

  4. Select the experiment file and use the Open button to load it in the Syrcadia software.

Running the Configuration

The configuration is already set up with a Zone containing two Alina fixtures, which correspond to the two light treatments used in the experiment: 1 (Less Blue) and 2 (More Blue).

When you first load the file, you will see a warning in a red box in the top right indicating the Scheduler is not running, which is needed to automatically activate and deactivate the lights at the appropriate times for the experiment.

There are multiple ways to resume the Scheduler. Is there a particular method you'd prefer we tell students to use?

To resume the schedule from the Dashboard, click the home icon and use either the Resume icon on the Zone 1 tile or Resume All in the bottom right.

You can also resume the schedule from the Zone 1 menu. From the Dashboard, click the Zone 1 tile.

From Settings, click the Zones tab and use the box arrow icon.

Once in the Zone 1 menu, if necessary, select the Scheduler tab. Use either of the Resume icons in the top right or bottom left to start the schedule.
