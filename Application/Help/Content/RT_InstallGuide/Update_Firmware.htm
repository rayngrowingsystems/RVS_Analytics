Update Firmware

RAYN Touch ships with Syrcadia software installed. If a new firmware version has been provided to you by RAYN, please follow the instructions below to update your controller.

  1. Copy the firmware update file on the root directory of a USB drive that has been formatted to FAT32. The file will be named similarly to ETC_RAYNTouch_#.#.#.#.fw, where # will be replaced with the version numbers.

Note: Please make sure the software file is not inside any other folder or sub-directory as this will hide it from the update function.

  1. With the unit powered on, insert the USB drive into the receptacle located on the top right side of the enclosure.
  2. Access the front of the touchscreen and navigate to Setup >Files >Advanced >Update Firmware. A confirmation message will display. Select “Yes” to continue with the update.

CAUTION: The update process may take a few minutes. Do not shutdown or remove power from the touchscreen until the update process has finished. When prompted, restart the touchscreen.

  1. When the touchscreen has booted, navigate to the Preferences tab in Settings to verify the updated software version number is displayed.
  2. Remove the USB drive from the unit.
